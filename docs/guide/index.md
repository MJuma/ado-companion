# Introduction

**ADO Companion** is a cross-browser extension (Chrome, Edge, Firefox) that adds extra functionality to [Azure DevOps](https://dev.azure.com).

It runs as a content script on `dev.azure.com` and legacy `*.visualstudio.com` organizations, injecting helpful UI directly into the pages you already use — without changing how Azure DevOps works.

## Highlights

- **Cross-browser** — one extension for Chrome, Edge, and Firefox.
- **Native look & feel** — built with [Fluent UI Web Components](https://github.com/microsoft/fluentui/tree/master/packages/web-components), themed to match Azure DevOps (light and dark).
- **Lightweight** — built with [SolidJS](https://solidjs.com) and [WXT](https://wxt.dev) for a small, fast bundle.
- **Isolated** — UI is injected via Shadow DOM so it never clashes with Azure DevOps styles.

## Get it

Head to the [Install guide](/guide/install) to add ADO Companion to your browser.
