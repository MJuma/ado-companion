What it adds

Three focused enhancements that live natively in the pull request UI. Each can be toggled on or off in settings.

📝

PR Markdown Review

A native “Review” view renders Markdown specs as a clean document with a Word-style comment rail synced to real Azure DevOps threads. Comment on the exact phrase you select, @mention teammates, paste images, and finally see Mermaid diagrams and syntax-highlighted code.

🧭

PR Activity Filter

Cut through the noise on the Overview timeline with filter tabs — Comments, Commits, Actions, and System Messages — so you can find what actually happened without scrolling past bot chatter.

🚦

PR Pipelines Tab

See every pipeline’s stages and jobs right inside the pull request — color-coded status circles that link straight to the logs, a live status badge on the tab, and auto-refresh, so you never leave the PR to check a build.

Install in a minute

Chrome & Edge

  1. Download the Chrome zip above and unzip it.
  2. Open chrome://extensions (or edge://extensions).
  3. Turn on Developer mode.
  4. Click Load unpacked and pick the folder.

Firefox

  1. Download the Firefox .xpi above.
  2. Open about:addons.
  3. Gear ⚙ → Install Add-on From File… → pick the .xpi.
  4. Confirm — it’s Mozilla-signed and auto-updates from here on.

Open any dev.azure.com pull request and the new views appear. Store listings (one-click install) for Chrome, Edge, and Firefox are planned.
